## Features

- **RESTful API** - Manage tour data with CRUD operations
- **User Authentication** - Secure routes with JWT-based authentication
- **Data Validation** - Ensure data integrity with Mongoose validation
- **Error Handling** - Centralized error handling for better debugging
- **API Documentation** - Comprehensive API documentation using Swagger

## Documentation

The API documentation is available [here](https://tours-app-api-drab.vercel.app/api-docs).

## TechStack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** JSON Web Tokens (JWT)
- **Deployment:** Vercel

## Getting_Started

```bash
# Install dependencies
npm install

# Start development server
npm start
```
